The modern web is filled with an increasing number of new technologies that change the way we create and implement our work. While they certainly help the development ecosystem to move forward, we believe that it's important to keep in mind the wunderful things that pure CSS can achieve, without needing to overengineer new solutions a lot of the time. A lightweight, fast, and amazing web is better for everyone, and we should strive for it, together.
